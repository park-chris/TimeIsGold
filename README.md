# TimeIsGold
> 타이머를 이용한 시간 측정 및 저장, 모니터링

</br>

## 1. 제작 기간 & 참여 인원
- 2022년 2월 1일 ~ 2022년 4월 23일
- 개인 프로젝트 (기획, 개발, 디자인 등)


</br>

## 2. 사용 기술 및 라이브러리
`Kotlin` `MVC` `MVVM` `AAC` `Room` `LiveData` `ViewModel` `Repository` `Figma` `Glide` `MPAndroidChart`

</br>

## 3.핵심 기능
#### 타이머 기능
- 타이머를 작동하고 기록으로 남길 수 있습니다.
#### 모니터링 기능
- 사용자는 일주일 동안의 각 항목별의 기록을 그래프로 확인할 수 있습니다.

</br>
